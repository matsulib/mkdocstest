# mkdocstest

## Installing
```bash
pip install -r requirements.txt
```

## Building the site
```bash
mkdocs build
```

## Deploying your docs on localhost
```bash
mkdocs serve
```

## Deploying your docs on GitHub Pages
```bash
mkdocs gh-deploy
```
